# 🏌️ 노스팜CC F&B 식음팀 프로모션 웹사이트

## 📋 프로젝트 개요
노스팜 컨트리클럽(NPCC, www.npcc.co.kr) 홈페이지의 **식음팀 음식 & 프로모션 홍보**를 위한 팝업 기반 웹사이트 프레임(틀).  
레스토랑 / 그늘집 / 스타트하우스 각 업장의 메뉴 및 이벤트를 시각적으로 홍보하는 것이 목적.

---

## ✅ 현재 구현된 기능

### 🎯 팝업 시스템 (3종)
| 팝업 | 내용 | 위치 |
|------|------|------|
| popup1 | 봄 시즌 스페셜 코스메뉴 (초록 테마) | 좌상단 |
| popup2 | 그늘집 봄 간식 세트 특가 (골드 테마) | 우상단 |
| popup3 | 얼리버드 조식 패키지 (다크 테마, 와이드형) | 하단 중앙 |

**팝업 기능:**
- ✅ 페이지 로드 0.8초 후 자동 오픈
- ✅ 오늘 하루 보지 않기 (localStorage 활용)
- ✅ 드래그 앤 드롭 이동 가능 (데스크탑)
- ✅ ESC 키로 전체 닫기
- ✅ 오버레이 클릭으로 전체 닫기
- ✅ 헤더 벨 아이콘으로 재오픈 가능
- ✅ 하단 플로팅 버튼으로 재오픈 가능

### 🏠 페이지 구성
- **헤더**: 상단 공지바 + 메인 네비게이션 (스크롤 시 상단바 숨김)
- **히어로 슬라이더**: 3개 슬라이드 자동 전환 (5초 간격)
- **Quick Info Bar**: 레스토랑/그늘집/카페 운영시간 한눈에 확인
- **프로모션 섹션**: 3종 프로모션 카드 (클릭 시 해당 팝업 오픈)
- **레스토랑 메뉴**: 탭 전환 (중식/조식/석식/일품요리)
- **그늘집/스타트하우스**: 배경 이미지 위 글래스모피즘 카드
- **공지 & 예약문의**: 공지사항 리스트 + 인라인 예약 폼
- **푸터**: 식음팀 연락처, 운영시간, SNS, 오시는 길

### 🎨 디자인 특징
- **브랜드 컬러**: 딥그린 (#1a3a2a) + 골드 (#c9a84c)
- **폰트**: Noto Serif KR (제목) + Noto Sans KR (본문)
- **반응형**: 데스크탑 → 태블릿(900px) → 모바일(680px) 3단계
- **애니메이션**: 스크롤 리빌, 슬라이더, 팝업 스프링 효과
